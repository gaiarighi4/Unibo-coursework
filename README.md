# Unibo Coursework

Archivio di alcuni esercizi, laboratori ed appunti svolti durante il mio percorso di studi in Informatica presso l'Università di Bologna.
Questa repo raccoglie esercitazioni minori a scopo di archivio personale. Per i progetti più significativi, guarda i repo pinnati sul mio profilo.

---

## Note

Questo materiale è pubblicato a scopo di archivio personale e di trasparenza sul percorso di studio. Il codice non è necessariamente ottimizzato o production-ready: molti esercizi sono soluzioni didattiche scritte per consolidare un singolo concetto, non progetti completi.
